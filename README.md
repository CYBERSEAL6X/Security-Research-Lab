# CYBERSEAL6X Threat Exposure Analyzer

A lightweight cybersecurity assessment tool that provides a rapid exposure score and executive-ready report in minutes.

Developed by **CYBERSEAL6X Security Research**  
https://cyberseal6x.com

---

## Overview

Most cyber incidents do not begin with advanced zero-day exploits.  
They begin with small exposures:

- Disabled protections  
- Weak configurations  
- Excessive privileges  
- Missing visibility  

This tool provides a quick snapshot of exposure indicators and generates a structured report to help prioritize remediation.

---

## Features

- Threat Exposure Score (0–100)
- Key risk indicators
- Executive summary
- Branded HTML report
- TXT and JSON output
- No installation required

---

## Quick Start (Run)

How to Run the Script (Step by Step)
Step 1 — Open PowerShell as Administrator

Click Start

Type PowerShell

Right-click Windows PowerShell

Click Run as administrator

Step 2 — Run this command

Copy and paste:

irm https://raw.githubusercontent.com/CYBERSEAL6X/Security-Research-Lab/main/ThreatExposure.ps1 | iex


Press Enter.

Step 3 — What will happen

The script will:

Download automatically

Run the security checks

Generate reports

Save them in:

C:\Users\Public\CYBERSEAL6X_Reports\

Step 4 — Open the Report

Open File Explorer

Go to:

C:\Users\Public\CYBERSEAL6X_Reports\


Open the HTML report
